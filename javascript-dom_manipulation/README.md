# JavaScript DOM Manipulation

This project contains basic JavaScript scripts that practice DOM manipulation, event handling, and fetching data using the Fetch API.

Each script interacts with HTML elements using `document.querySelector`, event listeners, and asynchronous requests.

---

## Tasks

### 0. Change header color
Updates the text color of the `<header>` element to red.

File: `0-script.js`

---

### 1. Click to change color
Changes the header color to red when the user clicks the element with id `red_header`.

File: `1-script.js`

---

### 2. Add class to header
Adds the CSS class `red` to the header when clicking `red_header`.

File: `2-script.js`

---

### 3. Toggle header color
Toggles the header class between `red` and `green`.  
The header always has exactly one class.

File: `3-script.js`

---

### 4. Add list item
Adds a new `<li>Item</li>` to the list `.my_list` when clicking `add_item`.

File: `4-script.js`

---

### 5. Update header text
Changes the header text to:

```
New Header!!!
```

when clicking `update_header`.

File: `5-script.js`

---

### 6. Fetch Star Wars character
Fetches a character from the Star Wars API and displays the name inside `#character`.

File: `6-script.js`



---

### 7. Fetch movie list
Fetches Star Wars movies and lists all titles inside `#list_movies`.

File: `7-script.js`

---

### 8. Translate “hello”
Fetches a translation of “hello” in French and displays it inside `#hello`.  
Works even when script is loaded in `<head>`.

File: `8-script.js`



---

## Concepts practiced

- DOM selection
- Event listeners
- Class manipulation
- Dynamic element creation
- Fetch API
- Promises
- DOMContentLoaded
- Asynchronous JavaScript
